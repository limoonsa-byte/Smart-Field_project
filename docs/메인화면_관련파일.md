# 메인 화면 관련 파일

## 1. **index.html**
- **메인 구조**: `#screen-main` div, 네 개 버튼(도면 그리기 / 도면 보관함 / 견적 내기 / 관리자 세팅)
- **head**: `window.goScreen` 최소 정의, GAS 배너 제거 스크립트, 에러 리스너
- **본문**: 메인 버튼 `onclick` + `data-goscreen`, 메인 전용 `click`/`touchend` 캡처 스크립트

## 2. **css.html**
- `#screen-main`: `pointer-events: auto`, `z-index`, 버튼 포인터/커서
- `.screen` / `.screen.active`: 화면 전환 표시
- GAS 배너 숨김용 선택자들

## 3. **js_main.html**
- `goScreen(screenId)`: 화면 전환 + draw/list/estimate/settings별 초기화
- `window.goScreen = goScreen` 할당

## 4. **기타**
- **js_drawing.html**, **js_history.html**, **js_estimate.html**, **js_settings.html**: 각 화면에서 `goScreen('main')` 호출

---
클릭이 안 될 때: **GAS 배너가 메인 위에 겹쳐 있으면** 배너가 클릭을 받음.  
→ 메인(`#screen-main`)을 배너보다 위 레이어로 올리기(z-index 상향).
