# 스마트 현장관리 (Smart Field)

Google Apps Script 웹앱 — 도면 그리기, 보관함, 견적, 일정, 관리자 세팅.

---

## 프로젝트 구조 (루트)

| 구분 | 파일 |
|------|------|
| **앱 실행** | `code.js`, `index.html`, `css.html`, `js_main.html`, `js_drawing.html`, `js_history.html`, `js_estimate.html`, `js_schedule.html`, `js_settings.html`, `download.html` |
| **PWA** | `pwa_index.html`, `manifest.json`, `sw.js`, `icons/` |
| **설정** | `appsscript.json`, `.clasp.json` |

**문서**는 모두 **`docs/`** 폴더에 있습니다.

- 작업 규칙, 새 컴퓨터 세팅, 코드 구조 등 → **`docs/`** 참고
